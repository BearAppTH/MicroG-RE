## [2.0.0] - 2025

### Added
- UI ออกแบบด้วย Material 3 Expressive
- รองรับ Google Account Authentication
- Package name `app.bear.android.gms` แทน `com.google.android.gms`
- รองรับแอปที่ผ่านการ patch แล้วโดยไม่ต้อง root
- Crowdin integration สำหรับการแปลภาษา
- GitHub Actions workflows สำหรับ build และ release อัตโนมัติ

### Changed
- Fork จาก [MorpheApp/MicroG-RE](https://github.com/MorpheApp/MicroG-RE)
- ปรับ branding เป็น BearAppTH
- อัพเดท dependencies เป็นเวอร์ชันล่าสุด
