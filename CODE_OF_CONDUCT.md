# Code of Conduct

## พันธสัญญาของเรา

เพื่อสร้างสภาพแวดล้อมที่เปิดกว้างและเป็นมิตร ผู้ดูแลโปรเจกต์และผู้ร่วมพัฒนา Bear MicroG
ให้คำมั่นว่าจะทำให้การมีส่วนร่วมในโปรเจกต์นี้ปราศจากการคุกคามสำหรับทุกคน
ไม่ว่าจะมีอายุ ขนาดร่างกาย ความพิการ เชื้อชาติ อัตลักษณ์ทางเพศ ระดับประสบการณ์
สัญชาติ รูปลักษณ์ภายนอก ศาสนา หรือรสนิยมทางเพศเช่นใดก็ตาม

## มาตรฐานของเรา

พฤติกรรมที่ช่วยสร้างสภาพแวดล้อมเชิงบวก:

- ใช้ภาษาที่เป็นมิตรและให้เกียรติผู้อื่น
- เคารพความคิดเห็นและประสบการณ์ที่แตกต่าง
- ยอมรับคำวิจารณ์เชิงสร้างสรรค์ด้วยใจเปิดกว้าง
- ให้ความสำคัญกับสิ่งที่ดีที่สุดต่อชุมชนและโปรเจกต์

พฤติกรรมที่ยอมรับไม่ได้:

- การใช้ภาษาหรือภาพที่มีนัยทางเพศ และการล่วงละเมิดในทุกรูปแบบ
- การกลั่นแกล้ง ดูหมิ่น หรือแสดงความคิดเห็นในเชิงดูถูก ทั้งส่วนตัวและสาธารณะ
- การเผยแพร่ข้อมูลส่วนตัวของผู้อื่นโดยไม่ได้รับอนุญาต
- พฤติกรรมอื่นใดที่ไม่เหมาะสมกับสภาพแวดล้อมทางวิชาชีพ

## ขอบเขตความรับผิดชอบ

ผู้ดูแลโปรเจกต์มีสิทธิ์และหน้าที่ในการลบ แก้ไข หรือปฏิเสธ comment, commit, code,
wiki edit, issue และ contribution อื่น ๆ ที่ไม่สอดคล้องกับ Code of Conduct นี้
รวมถึงมีสิทธิ์ระงับสิทธิ์การมีส่วนร่วมของผู้ที่มีพฤติกรรมไม่เหมาะสมเป็นการชั่วคราวหรือถาวร

## ขอบเขตการบังคับใช้

Code of Conduct นี้มีผลทั้งในพื้นที่ของโปรเจกต์ (เช่น GitHub issues, pull requests,
discussions) และในพื้นที่สาธารณะเมื่อบุคคลนั้นเป็นตัวแทนของโปรเจกต์หรือชุมชน

## การรายงาน

หากพบพฤติกรรมที่ไม่เหมาะสม สามารถรายงานได้ผ่านช่องทางเดียวกับการรายงาน
security vulnerability ที่ระบุใน [SECURITY.md](SECURITY.md) หรือเปิด private
report ผ่าน GitHub ทีมผู้ดูแลจะตรวจสอบและตอบกลับทุกกรณีอย่างเป็นความลับ

## ที่มา

เอกสารนี้ดัดแปลงจาก [Contributor Covenant](https://www.contributor-covenant.org/th/version/1/4/code-of-conduct/) เวอร์ชัน 1.4
